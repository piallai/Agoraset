XCODE 4 Users

 

Mex is incompatible with Xcode 4

One of our reader use C code and mex files


Unfortunately, Xcode 4 is incompatible with mex


Therefore, given this link, we suggest the users the following process


----------------------------------------------------------------


1 - As a root user, go to the folder

~username/.matlab/R20XXa/


2 - Run the command

mex -setup


or the command (if mex is already a tex command)

/Applications/MATLAB_R20XX/bin/mex -setup


3 - Edit the generated file mexopts.sh


4 - Change gcc-4.0 to gcc in both maci and mac64i sections.


----------------------------------------------------------------
