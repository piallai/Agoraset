Demo with the scene 3, normal velocity

 

1 - Download data and environment parameters, put them in a data/ folder

2 - SET ENVIRONMENT VARIABLES & Get PARAMETERS (mesh, environment, population)


clear;


% Example with the scene 3 (Two obstacles) with normal speed


%----------------------------------------------------

% DEFINITION OF FOLDER NAMES

%----------------------------------------------------


% Environment data in folfer "Environment03"

% position data in 'scene_03_x1-25Hz' folder

environment_folder='data/Environment03/';

scene_folder='data/scene_03_x1-25Hz/';



%----------------------------------------------------

% GET PARAMETER OF MESH, ENVIRONMENT AND POPULATION

%----------------------------------------------------

meshing = classMeshing(strcat(environment_folder,'meshing'));

environment = classEnvironment(meshing,strcat(environment_folder,'environment03'));

population = classPopulation(strcat(scene_folder,'population01'));

environment_r2 = classEnvironment(environment,2);


%----------------------------------------------------

% GET KEY INFORMATIONS

%----------------------------------------------------

%The velocity is directly available :

velocity = population.velocities;


% compute the density map from population positions at

% time 500 with a gaussian kernel of 0.3 std deviation

ntime = 500;

std_density = 0.3;


density_map = environment.meshing.compute_density(population.positions, std_density, ntime);
