How to exploit the dataset

 

We present here some key operations to simply deal, under matlab, with the available position files and environment informations



Readers


The matlab readers contain the following key files


  1. -classPopulation.m : many interesting functions enabling to extract various quantities of interest (velocity, positions, density, ...)

  2. -classEnvironment.m : informations about the environment related to each scene

  3. -classMeshing.m : informations related to the meshes
